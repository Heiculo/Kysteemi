Mini projects for the course "Operating Systems and System Programming" course

Project 1: Warmup to C, Basic C program that reads data from an input file and then prints it out in a reverse.

Project 2: Unix Utilities, Simple versions of few commonly used commands. Includes cat, grep, zip and unzip. 

Project 3: Unix shell, Simple Unix shell that runs few basic commands and has support for parallel commands and redirection


All of the projects have been done in Fall of 2025
